最小圆覆盖衍生问题

给你二维平面上个点，求至少能覆盖其中个点的圆中最小圆的半径。

题解

小学奥数好题

考虑暴力。因为三点定圆，所以我们枚举3个点，然后再判断这个圆覆盖的点的数量。

期望时间复杂度

抱歉，这是一个错误算法。显然有些数据存在更小的圆。

这种情况大多是两点在圆周上，另一点在两点连线上。

直接暴力处理这种情况，期望，然后就可以再按照上面说的做。

考虑更加优秀的算法。

不难发现，半径具有二分性，所以我们考虑二分半径。

然后此问题就变成了给你平面上个点，再给你一个半径为的圆，求其能覆盖的点的数量。

显然此问题可以求

首先我们发现，覆盖的点最多的数量为面积覆盖最多的次数。

然后覆盖最多的面积的次数可以转化为极角对应弧的覆盖次数。

然后点的覆盖次数就是弧覆盖了。

我们对于圆交点所成弧求其极角，再极角排序，然后将圆拉直，求线段覆盖次数。

然后就好了。

对每个圆要循环，极角排序，check函数复杂度就为了

为了优化二分上界，考虑一个定理：

定理：最小圆覆盖中，最小圆覆盖的所有点中的最远点对的距离一定大于此圆的半径。

这题如果取则变成最小圆覆盖问题，半径上界为最远点对的距离，

所以我们二分上界取即可。

所以这个貌似是现在最优解。

#pragma GCC optimize(2)
#pragma GCC optimize(3)
#pragma GCC optimize("Ofast")
#include <bits/stdc++.h>
#define Mn 505
using namespace std;
const double eps = 1e-10;
const double pi = acos(-1.0);
#define sqr(x) ((x) * (x))
double Fmax(double a,double b){return a > b ? a : b;}
double RF;
int n,k;
struct point
{
    double x,y;
    void read(){scanf("%lf%lf",&x,&y);}
    void print(){printf("%lf%lf\n",x,y);}
    double friend dis(const point &a,const point &b) { return sqrt(sqr(a.x - b.x) + sqr(a.y - b.y));}
} p[Mn + 5];
struct alpha
{
    double v;
    bool flag;
    bool friend operator < (const alpha &a,const alpha &b){ return a.v < b.v;}
}alp[Mn * Mn + 5];
int check()
{
    int MAX = 0;
    for(int i = 0; i < n;i++)
    {
       int t = 0;
       for(int j = 0; j < n;j++)
       {
           if(i == j)
              continue;
           double theta,phi,D;
           D = dis(p[i],p[j]);
           if(D > 2.0 * RF) continue;
           theta = atan2(p[j].y - p[i].y,p[j].x - p[i].x);
           if(theta < 0)
              theta += 2 * pi;
           phi = acos(D / (2.0 * RF));
           alp[t].v = theta - phi + pi * 2;
           alp[t].flag = true;
           alp[t + 1].v = theta + phi + pi * 2;
           alp[t + 1].flag = false;
           t += 2;
       }//求极角
       sort(alp,alp + t);//极角排序
       int sum = 0;
       for(int j = 0;j < t;j++)
       {
           if(alp[j].flag) sum++; else sum--;
           MAX = max(MAX,sum);
       }//弧覆盖
    }
   // cout<<MAX+1<<endl;
    return MAX + 1;
}
int main() {
    double Mdist = 0;
    scanf("%d%d",&n,&k);
    for(int i = 0;i < n;i++)
       p[i].read();
    for (int i = 0; i < n; i++)
      for (int j = 0; j < n; j++)
        if (i != j)
          Mdist = Fmax(Mdist,dis(p[i],p[j]));
//  cout<<Mdist<<endl;
    random_shuffle(p,p + n);//自动忽略，卡常
    double L = 0;
    double R = Mdist;//二分上界取最远距离
    while (fabs(R - L) > eps)
    {
        double mid = (L + R) / 2;
        RF = mid;
        //cout<<RF<<endl;
        if (check() >= k)
          R = mid; else
        L = mid;
    }//二分
    printf("%.8lf\n",L);//精度
    return 0;
}
